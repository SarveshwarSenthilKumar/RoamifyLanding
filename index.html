About Roamify

Roamify is an AI-powered web application that helps you discover outdoor activities near you. Whether you're into hiking, biking, or exploring local events, Roamify provides tailored suggestions based on your location, preferences, and the current weather. Let Roamify plan your next adventure!

This project is built using Flask, with modular architecture to ensure clean, scalable, and rapid development. The app is designed to help you discover new places and activities, making outdoor exploration easier and more enjoyable.

Installation

To get started with Roamify, follow these steps:

  • Clone the repository: git clone https://github.com/yourusername/Roamify.git
  • Create a virtual environment: python -m venv venv
  • Activate the virtual environment: source venv/bin/activate (or venv\Scripts\activate on Windows)
  • Install the dependencies: pip install -r requirements.txt
  • Set up environment variables in the .env file
  • Run the application: python run.py
